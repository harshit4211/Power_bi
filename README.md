# Sales Dashboard using Power BI

## Project Overview

This project presents an interactive Sales Dashboard built in Microsoft Power BI to analyze business performance across different regions, product categories, and time periods. The dashboard enables users to monitor key business metrics and identify sales trends through dynamic visualizations.

---

## Objectives

- Analyze overall sales performance.
- Track profit and sales trends over time.
- Compare sales across different regions.
- Evaluate product category performance.
- Monitor total quantity sold.
- Provide an interactive dashboard for business decision-making.

---

## Dataset

The dashboard is built using an **Orders** dataset containing transactional sales information.

### Key Fields
- Order Date
- Region
- Product Category
- Sales
- Profit
- Quantity Ordered

---

## Dashboard Features

The dashboard includes the following visualizations:

- **Sales by Region** (Bar Chart)
- **Sales Trend by Month** (Line Chart)
- **Sales by Product Category** (Pie Chart)
- **Quantity Ordered by Product Category** (Clustered Column Chart)
- **Total Sales KPI**
- **Total Profit KPI**
- **Total Quantity Ordered KPI**
- Interactive filters and navigation button

---

## Key Insights

- Compare regional sales performance.
- Identify top-performing product categories.
- Analyze monthly sales trends.
- Monitor overall profit generated.
- Evaluate product demand through quantity sold.
